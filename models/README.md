The first step to running this pipeline is to copy the models from the SNACKKSS_NLP pipeline into this directory. Specifically, from the "output" folder, copy the following direcotries into this one:
gene_study_model
gene_sample_model
gene_target_model
gene_control_model
drug_study_model
drug_sample_model
drug_target_model
drug_control_model

The reason SNACKKSS_NLP is a separate repository is that it is designed to be run only once, while this one is designed to update itself continually.
